# Ovumcy v0.6.0

Release date: 2026-03-15

## Highlights

- Added owner-only cycle factor tracking for irregularity context in daily logs, exports, and conservative stats hints.
- Refined sparse-data insights so early cycle history no longer over-promises prediction confidence.
- Cleaned up owner-facing settings and dashboard UX, including clearer destructive copy and less redundant logging controls.
- Added a privacy-safe hero demo asset pack and install-prompt capture for README and release materials.

## Security and quality

- Irregular-cycle dashboard copy is now more cautious and no longer implies a precise ovulation date when recent data is sparse.
- Owner-only cycle factors remain sanitized outside the supported owner browser path across UI and export flows.
- CI and coverage publication were tightened so Codecov patch checks publish reliably and stay aligned with the release branch state.
- Browser, service, and HTML regression coverage was extended around stats, settings, and owner day logging behavior.

## Upgrade notes

- No database migration is required for this release.
- Existing deployments can upgrade in place and pin `OVUMCY_IMAGE=ghcr.io/terraincognita07/ovumcy:v0.6.0` if they do not want to track `latest`.
- No auth/session contract changes are required for operators in this release.
